# Unheard Voices

Interactive React + Vite visualization highlighting underrepresented historical events.

## Quick Start

1. Install dependencies:
	npm install
2. Start dev server:
	npm run dev
3. Build for production:
	npm run build

## If "nothing works"

If you see `vite: not found` or an exit code `127`, it usually means dependencies were not installed.

Run:

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

This project expects Node.js `>=18.18.0`.
